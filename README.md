## Quick Start

Running the app

1. install dependencies

   ```bash
   yarn install
   ```

2. start a hardhat node

   ```bash
   yarn chain
   ```

3. run the app, `you'll need to open a new command prompt`

   ```bash
   yarn contracts:build
   yarn deploy
   yarn start
   ```
